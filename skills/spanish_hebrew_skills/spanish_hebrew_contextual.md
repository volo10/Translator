# Contextual Understanding Skill: Spanish to Hebrew

## Overview
Deep comprehension of Spanish text meaning using context for accurate Hebrew translation.

## Key Context Challenges: Spanish to Hebrew

### 1. Definiteness Interpretation
Spanish articles (el/la) → Hebrew ה- prefix
- "un libro" (indefinite) → "ספר" (no marker)
- "el libro" (definite) → "הספר" (with ה)

### 2. Tense/Aspect Resolution
Spanish imperfect/preterite → Hebrew past
- "hablaba" (ongoing) → context needed
- "habló" (completed) → past tense
Hebrew has less aspectual distinction

### 3. Ser vs Estar Context
Both map to Hebrew היה/הוא
- "estar" (temporary) → present participle in Hebrew
- "ser" (permanent) → simple form
Context determines Hebrew construction

### 4. Gender Context
Spanish gender ≠ Hebrew gender
- "la mesa" (f) → "השולחן" (m)
Context helps determine if gender matters semantically

### 5. Formality Level
Spanish tú/usted → Hebrew less formal distinction
- Detect formality from context
- Apply through word choice, not pronouns

### 6. Ambiguity Resolution
**Word Sense:**
- "banco" (bank/bench) - context determines
- Hebrew: בנק (bank) or ספסל (bench)

**Pronoun Reference:**
- "él/ella" clear in Spanish
- Hebrew requires same: הוא/היא
- Context resolves ambiguous references

### 7. Idiomatic Recognition
Detect Spanish idioms for Hebrew equivalents:
- "llover a cántaros" → "ירד גשם כמו מטר" or paraphrase
- Context shows non-literal meaning

### 8. Temporal Context
Spanish tense nuances → Hebrew time markers
- "he comido" vs "comí" → both "אכלתי"
- Add time markers: כבר (already), עכשיו (now)

### 9. Pragmatic Meaning
Indirect speech acts:
- "¿Puedes cerrar la ventana?" (request)
- Hebrew: "אתה יכול לסגור את החלון?" (same structure)
- Politeness markers differ

### 10. Ellipsis Recovery
Spanish omits more:
- "Como manzanas" (subject omitted)
- Hebrew prefers explicit: "אני אוכל תפוחים"
- Recover omitted information from context

## Application Strategy
1. Identify all context layers
2. Resolve ambiguities before translation
3. Determine Hebrew structures based on context
4. Preserve pragmatic meaning in Hebrew
5. Use context to choose appropriate Hebrew register
