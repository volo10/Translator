# Error Correction: Hebrew to English

## Hebrew-Specific Error Patterns

### 1. Missing Nikud (Vowel Points)
Modern Hebrew typically omits nikud
- Context determines pronunciation and meaning
- כתב could be: katav (wrote), k'tav (writing), kotev (writes)
- Usually not an error, standard practice

### 2. Keyboard Layout Errors
**Hebrew keyboard specific:**
- ח/כ confusion (adjacent keys)
- ו/ז confusion
- ש/ס confusion (both "s" sounds)
- Final forms (ך, ם, ן, ף, ץ) sometimes typed as regular

### 3. Common Spelling Errors

**Phonetic Confusion:**
- ס/ש (both "s" sounds in some contexts)
- כ/ק (k sounds)
- ת/ט (t sounds)
- א/ע (both silent/glottal in modern pronunciation)

**Example:**
- שרות vs סרות (services - ש vs ס)
- מכונית vs מקונית (car - כ vs ק)

### 4. Missing Definite Article
Casual writing may omit ה-
- ספר vs הספר (book vs the book)
- Usually intentional in informal text

### 5. Wrong Prefix Combination
- ב+ה should become בַּ (ba-)
- ל+ה should become לַ (la-)
- מ+ה should become מֵ (me-)
- Error: writing separately when should combine

### 6. Gender Agreement Errors
- Verb doesn't match subject gender
- Adjective doesn't match noun gender
- "היא אומר" → should be "היא אומרת" (she says)

### 7. Slang and Abbreviations
**Very common in Hebrew:**
- נ.ב. (postscript)
- וכו' (etc.)
- מס' (number)
- אמר"י (America/American - אמריקה)
- Recognize as intentional, not errors

### 8. English Loan Word Spelling
Hebrew borrows many English words
- May be spelled phonetically in Hebrew letters
- Multiple valid spellings possible
- קומפיוטר / מחשב (computer)
- אימייל / דוא"ל (email)

### 9. Arabic-Origin Words
Israeli Hebrew includes Arabic words
- אחלה (akhla) - great
- יאללה (yalla) - let's go
- סבבה (sababa) - cool/okay
- Standard usage, not errors

### 10. Missing Spaces
Hebrew can run words together
- Especially with prefixes
- בביתשלי vs בבית שלי (in my house)
- Context needed to separate

## Error Detection Strategy

### Priority 1: High-Impact Errors
- Wrong binyan (changes meaning significantly)
- Gender mismatch affecting comprehension
- Missing critical prepositions

### Priority 2: Medium-Impact Errors
- Phonetic spelling variations
- Prefix combination errors
- Article omissions

### Priority 3: Low-Impact Errors
- Stylistic variations
- Valid alternative spellings
- Intentional informal spelling

## Context-Based Correction

### Homograph Resolution
Without nikud, many forms look identical:
- כתב: katav (wrote), ktav (writing), kotev (writes)
- Context determines correct reading

**Example:**
"הוא כתב מכתב"
- First כתב = katav (wrote) - past tense
- Second מכתב = mikhtav (letter) - noun

### Gender Context
Hebrew gender visible in verbs/adjectives
- Use verb endings to determine subject gender
- מדבר (m) vs מדברת (f)
- Informs pronoun choice in English

## Special Cases

### Acronyms
Hebrew has many acronyms:
- צה"ל (Tzahal - IDF)
- רמ"ח (Ramatkal - Chief of Staff)
- ת"א (Tel Aviv)
- Don't "correct" these

### Construct State (Smichut)
Special noun compound form:
- בית ספר (beit sefer - school, lit. house of book)
- Not an error, grammatical structure

### Text Message Hebrew
Heavily abbreviated, mixed with English
- "את רוצה 2 לבוא?" (Do you want to come?)
- Number "2" = ל (le-, to)
- Recognize as intentional informal style

## Correction Workflow

1. **Identify text type:** Formal, informal, SMS, etc.
2. **Check for ambiguous forms:** Resolve with context
3. **Validate gender agreement:** Match verbs to subjects
4. **Verify prefix combinations:** Correct if malformed
5. **Preserve slang:** Don't "correct" intentional informal language
6. **Note ambiguities:** Flag when multiple readings possible

## Translation Impact

### High-Impact Corrections
Affect English meaning significantly:
- Wrong binyan → wrong English verb
- Gender errors → wrong pronoun (he/she)
- Missing negation (לא)

### Low-Impact Corrections
Don't affect English translation much:
- ס/ש variations (both → "s")
- Missing definite article (context provides)
- Informal spelling (meaning clear)

## Best Practices

1. **Understand binyanim:** Critical for correct meaning
2. **Use context heavily:** Hebrew omits much that English needs
3. **Don't overcorrect slang:** Part of modern Hebrew
4. **Recognize loan words:** May have multiple valid spellings
5. **Check gender carefully:** Affects English pronouns
6. **Preserve register:** Informal Hebrew → informal English
7. **Note ambiguities:** When context doesn't resolve fully
8. **Accept variations:** Modern Hebrew has acceptable spelling flexibility
